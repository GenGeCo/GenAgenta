## TOOL PRINCIPALI
- `call_api` → chiama API REST
- `geocode_address` → coordinate da indirizzo
- `map_fly_to` → sposta mappa
- `map_select_entity` → seleziona entità
- `query_database` → query SQL
- `read_file` → leggi documentazione
- `get_user_actions` → cosa ha fatto l'utente (click, selezioni)
